This was another group project I worked on during my degree. We were tasked with building a system using Laravel — a PHP framework I had just started learning at the time. I decided to base the project on Jilid Enam, a coffee shop in Section 6, Shah Alam, where I was working part-time as a barista.

For the UI, I drew inspiration from Pinterest and personally took all the photos used in the project. Together with my team, we built an ordering system aimed at improving order flow and efficiency for customers. One key feature we added was a custom order ID generator to help keep track of individual orders.
